## ⚙️ Installation
1. Clone the repo:
   ```sh
   git clone https://github.com/Aniket-373/stoic-task.git
   cd my-app
   ```
2. Install dependencies:
  ```sh
  npm install
  ```
3. Run locally:
  ```sh
  npm start
  ```
4. Build for production:
  ```sh
  npm run build
  ```
